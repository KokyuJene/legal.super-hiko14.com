認証システム – 利用規約

最終更新日：2026年3月12日

1. 目的

1-1. 本規約は、auth.super-hiko14.com で提供される Discord 認証システム(以下「本認証」)の利用規約を定めるものです。
1-2. 本認証を利用するすべての方(以下「ユーザー」)は、本規約に同意したものとみなされます。
1-3. 本認証は、Discord サーバーへの入室資格を検証するためのシステムであり、年齢・ボット検出・VPN 利用を確認しています。

2. 認証の要件

2-1. ユーザーが本認証を通じて Discord サーバーへの入室を希望する場合、以下の要件を満たす必要があります。

2-1-1. Discord アカウント要件
 ・ 有効な Discord アカウント(メールアドレス確認済み)であること
 ・ アカウント作成から 14日以上経過していること
 ・ Discord 利用規約/コミュニティガイドラインを守っていること

2-1-2. ネットワーク要件
 ・ VPN/プロキシサービスを使用していないこと
 ・ 正常なインターネット接続を有していること

2-1-3. ボット検出要件
 ・ Cloudflare Turnstile による人間確認(チェックボックス)に応答すること
 ・ ボット/自動化ツールでないこと

3. 認証フロー

3-1. ユーザーが Discord 認可画面から認証を開始すると、以下のステップが実行されます。

 ① Discord OAuth 認可(ユーザーが許可ボタンをクリック)
 ② Cloudflare Turnstile によるボット検出
 ③ VPN/プロキシ検出(ProxyCheck API)
 ④ Discord アカウント年齢確認
 ⑤ 同一 IP からの複数アカウント検出
 ⑥ ロール付与/リダイレクト

3-2. 上記いずれかで失敗した場合、ユーザーは拒否理由とともにエラー画面へリダイレクトされます。
3-3. すべての検証に合格した場合、ユーザーに対して Discord サーバーでのロールが付与されます。

4. 禁止事項 / 認証拒否理由

4-1. 以下に該当するユーザーは認証が拒否される場合があります。

4-1-1. ボット判定
 ・ Turnstile で「非人間」と判定された場合
 ・ 複数回の認証失敗が発生した場合

4-1-2. VPN/プロキシ使用
 ・ VPN サービス(ExpressVPN, NordVPN 等)を使用している場合
 ・ プロキシサーバーを経由している場合
 ・ 匿名化/リレーサービスを使用している場合

※ 正規の ISP により提供される接続方法は除外されることがあります


4-1-3. アカウント年齢不足
 ・ Discord アカウント作成から 14日未満である場合

4-1-4. 複数アカウント検出
 ・ 同一 IP から複数の異なる Discord アカウントで認証した場合
 ・ 既にブロックされているアカウントと同じ IP を使用している場合

4-1-5. その他
 ・ 技術的エラー/サーバー側の問題が発生した場合
 ・ 運営が不適切と判断する活動が検出された場合

5. 複数アカウント利用について

5-1. 本認証では、同一 IP からの複数アカウント認証を検出し、後発のアカウントをブロックします。
5-2. 複数アカウントの使用は以下の理由から制限されています。
 ・ 不正アクセス/乗っ取り検出
 ・ 複数規約違反対策
 ・ コミュニティの健全性管理
5-3. 正当な理由がある場合(家族での利用など)は、お問い合わせフォームより申し立てることができます。

6. ブロック後の対応

6-1. 認証に失敗したユーザーは、failure reason を確認してから再度認証に進むことができます。
6-2. 技術的な誤りによるブロックまたは異議がある場合は、説明とともにお問い合わせフォームから申し立てることができます。
6-3. 申し立てについては、運営が確認した上で判断し、その結果を返答します。返答には時間がかかる場合があります。
6-4. VPN/プロキシの使用を中止して再認証することで、多くの場合 ブロックは解除されます。

7. 免責事項

7-1. 本認証の提供者は、以下については一切の責任を負いません。
 ・ 誤った判定(ボット判定エラー、年齢検証エラー等)
 ・ 外部サービス(Discord, Cloudflare, ProxyCheck)の不具合
 ・ ネットワーク遅延/タイムアウト
 ・ 本認証の利用に伴い生じたあらゆる損害

7-2. VPN/プロキシ検出の精度は 100% ではありません。検出漏れ/誤検出が発生することがあります。
7-3. 本認証は誰でも自由に提供/変更/中止できるものとします。
7-4. 本認証は Discord/Cloudflare/ProxyCheck とは一切関係がなく、これらの企業は本認証に関して責任を負いません。

8-1. 本規約は必要に応じて予告なく変更できるものとします。
8-2. 変更後の規約は本ページへの掲載をもって効力を生じるものとします。
8-3. 変更後も本認証を継続して利用した場合、ユーザーは変更後の規約に同意したものとみなされます。
