Privacy.

認証システム – プライバシーポリシー

最終更新日：2026年3月12日

本プライバシーポリシーは、auth.super-hiko14.com で提供される Discord 認証システム(以下「本認証」)における情報の取り扱いについて定めるものです。本認証を利用することで、本ポリシーに同意したものとみなされます。

1. 収集する情報

本認証では、以下の情報を収集します。

1-1. Discord より取得する情報
 ・ Discord ユーザーID
 ・ Discord ユーザー名
 ・ Discord メールアドレス(プライベート スコープで同意した場合)
 ・ Discord アカウント作成日時(年齢検証用)

1-2. ネットワーク情報
 ・ アクセス元の IP アドレス
 ・ リクエストのタイムスタンプ

1-3. ボット検出情報
 ・ Cloudflare Turnstile の検証結果(ボット判定のみ)
 ・ IP アドレスに紐づく VPN/プロキシ検出結果

本認証はアカウント機能/ユーザープロフィール機能を有しておらず、メールアドレス等の詳細情報は基本的に収集しません。

2. 情報の利用目的

収集した情報は以下の目的のみで使用します。

 ・ Discord サーバーへの入室認証
 ・ ボット/自動化ツール による不正入室の防止
 ・ VPN/プロキシを介した接続の拒否
 ・ アカウント年齢要件の検証(14日以上)
 ・ 同一 IP からの複数アカウント接続の検出
 ・ 認証ログの記録と監視

収集した情報を上記以外の目的で使用することはありません。また、第三者への提供/販売も行いません。ただし、法令に基づく開示要求があった場合はこの限りではありません。

3. 情報の保持期間

本認証で収集した情報は以下の期間で保持されます。

3-1. IP アドレス/ログ情報
 認証日から 30日間、Supabase データベース上に保持されます。期限を過ぎたデータは自動削除されます。

3-2. Turnstile 検証結果
 Cloudflare の処理後、本サーバー側には記録されません(一時的なメモリのみ)。

3-3. Cookie 情報
 認証フロー中の セッション Cookie は認証完了後(最大 5分)に自動削除されます。

4. 外部サービスとの連携

本認証では以下の外部サービスを利用しています。これらの外部サービスに関するデータ処理については、各サービスのプライバシーポリシーが適用されます。

4-1. Discord OAuth 2.0
 ユーザー認証のため Discord から情報を取得します。詳細は Discord プライバシーポリシーをご確認ください。

4-2. Cloudflare Turnstile
 ボット検出のため Cloudflare Turnstile を使用しています。IP アドレスは Cloudflare に送信されます。詳細は Cloudflare プライバシーポリシーをご確認ください。

4-3. ProxyCheck.co
 VPN/プロキシ検出のため ProxyCheck API を使用しています。IP アドレスは ProxyCheck に送信されます。詳細は ProxyCheck プライバシーポリシーをご確認ください。

4-4. Vercel
 認証エンドポイントは Vercel Functions でホストされています。Vercel の詳細ポリシーについて、詳細は Vercel プライバシーポリシーをご確認ください。

5. ユーザーの権利

ユーザーは以下の権利を有しています。

5-1. アクセス権
 本認証で保有されている自分の情報にアクセスする権利があります。

5-2. 削除請求権
 保有している個人情報の削除を請求することができます(ただし、ログ保持期間中のみ)。

5-3. 異議申し立て
 VPN/ボット判定による拒否に異議がある場合は、お問い合わせフォームから申し立てることができます。

これらの請求については、本ページ下部の「お問い合わせ」より連絡してください。

6. セキュリティ対策

本認証で取得した情報は以下の方法で保護しています。

 ・ HTTPS による暗号化通信
 ・ HttpOnly Cookie による セッション管理
 ・ CSRF 防止(State パラメータ検証)
 ・ リプレイアタック防止(セッション Cookie の自動削除)
 ・ Supabase による暗号化保存

ただし、インターネット上の通信は完全に安全であることは保証されません。

7. お問い合わせ

本ポリシーに関するお問い合わせ・データ削除請求・異議申し立ては以下の方法にてご連絡ください。

 / Googleフォーム：お問い合わせページ
 / メール：kokyujene.contact@gmail.com

8. ポリシー変更

本ポリシーは必要に応じて予告なく変更される場合があります。変更後のポリシーは本ページへの掲載をもって効力を生じるものとします。最終更新日を必ずご確認ください。
